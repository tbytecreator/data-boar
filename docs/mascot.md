## Where they are used in the project

- **Favicon (browser tab):** `base.html` → `data_boar_mascote_color_translucent.png`. Optional: add `api/static/favicon.ico` (16×16/32×32) for legacy browsers.
- **About page:** `about.html` → color SVG (64×64) next to the app name.
- **Dashboard / Reports / Config / Help:** Each page uses the color SVG (40×40) in the main heading and (32×32) in the attribution block at the bottom.
- **README (GitHub):** `![Data Boar](api/static/mascot/data_boar_mascote_color.svg)` (EN and pt-BR).
- **Docs:** Use the same SVG or PNG where the Data Boar logo is shown (e.g. [PLAN_LOGO_AND_NAMING.md](PLAN_LOGO_AND_NAMING.md)).
- **Excel report:** The Report info sheet includes a small mascot (48×48 px) at D1; generated by `report/generator.py` from `data_boar_mascote_color_full.png`.
- **Heatmap PNG:** The sensitivity/risk heatmap image includes a small mascot in the lower-right corner (8% size); same source PNG.
- **Heatmap data sheet:** The same heatmap PNG is embedded in the Excel "Heatmap data" sheet below the raw data table, scaled to fit one page (letter/A4) when printed.

## Favicon

`api/static/favicon.ico` is generated from the mascot PNG (16×16 and 32×32). To regenerate after changing the mascot:

```bash
uv run python scripts/build_favicon.py
```

## Manual conversion (if you prefer)

- **Inkscape (Windows):** Open `data_boar_mascote_color.svg` → File → Export PNG Image → set width/height 32 → Export. Then in GIMP: open the PNG → Image → Scale to 32×32 and 16×16, export as `.ico` (GIMP can save as ICO with multiple layers/sizes), or use a site/tool to combine two PNGs into one ICO.
- **GIMP:** Open `data_boar_mascote_color_translucent.png` → Image → Scale Image to 32×32; duplicate layer, scale to 16×16; File → Export As → `favicon.ico` (choose ICO and include both sizes if the export dialog allows).
- **ImageMagick (Linux):** `convert api/static/mascot/data_boar_mascote_color_translucent.png -resize 32x32 -resize 16x16 api/static/favicon.ico` (or use `magick` and create 16 and 32 in one ICO: see `magick convert -help` for multi-size ICO). Simpler: `magick api/static/mascot/data_boar_mascote_color_translucent.png -define icon:auto-resize=16,32,48 api/static/favicon.ico`.
