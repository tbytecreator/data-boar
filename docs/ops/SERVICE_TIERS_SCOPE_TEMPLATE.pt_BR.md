# Template de escopo por tier de serviço (starter, growth, enterprise)

**English:** [SERVICE_TIERS_SCOPE_TEMPLATE.md](SERVICE_TIERS_SCOPE_TEMPLATE.md)

Use este template para definir tiers de serviço sem overpromise jurídico.

## Tier A — Starter (SMB)

- Objetivo principal: visibilidade rápida e primeiro backlog de remediação.
- Escopo típico: conectores limitados, janelas de varredura controladas.
- Entregáveis: relatório de achados, lista de prioridades, primeiro plano de ação.
- Exclusões: promessa de certificação, conclusões jurídicas.

## Tier B — Growth (médio porte)

- Objetivo principal: governança repetível e coordenação entre áreas.
- Escopo típico: múltiplos domínios de dados, revarreduras periódicas, tendência.
- Entregáveis: pacote de evidências, cadência de remediação, checkpoints de governança.
- Exclusões: atestado jurídico, substituição de auditoria completa SOC/ISO.

## Tier C — Enterprise Program

- Objetivo principal: modelo operacional escalável e suporte de evidência para assurance.
- Escopo típico: ambientes segmentados, controles mais estritos, expansão por fases.
- Entregáveis: pacotes estruturados de evidências, apoio ao mapeamento de controles, roadmap de maturidade.
- Exclusões: emissão formal de certificação, substituição de parecer jurídico.

## Frase obrigatória em todos os tiers

"Data Boar apoia descoberta técnica, geração de evidências e priorização de remediação. Interpretação jurídica e atestação de conformidade permanecem sob governança e assessoria jurídica do cliente."
