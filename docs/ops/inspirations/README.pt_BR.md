# Baseline de inspirações (security/GRC)

**English:** [README.md](README.md)

Objetivo: manter uma lista pequena e de alto sinal com fontes externas que inspiram decisões de hardening e roadmap, sem virar política obrigatória.

Princípio:

- Recomendações externas são **insumo**, não ordem.
- Decisões de produto precisam ser validadas no nosso contexto, testes e segurança de rollout.

Conjunto inicial:

- [Security Now / GRC](SECURITY_NOW.md)
- [Projetos e guias OWASP](OWASP.md)
- [CISA KEV + advisories](CISA_KEV_AND_ADVISORIES.md)

Como usar esta pasta:

1. Ler a nota da fonte.
2. Extrair uma ação candidata.
3. Decidir: adotar agora, backlog, ou rejeitar.
4. Ligar a decisão a um artefato do repositório (plano/doc/teste/workflow).

Mantenha esse baseline enxuto. Adicione novas fontes só quando forem úteis de forma recorrente.
