# Security/GRC inspiration baseline

**Português (Brasil):** [README.pt_BR.md](README.pt_BR.md)

Purpose: keep a small, high-signal list of external sources that inspire roadmap and hardening decisions without becoming mandatory policy.

Principle:

- External recommendations are **inputs**, not orders.
- Product decisions must be validated against our context, tests, and rollout safety.

Current starter set:

- [Security Now / GRC](SECURITY_NOW.md)
- [OWASP projects and guidance](OWASP.md)
- [CISA KEV + advisories](CISA_KEV_AND_ADVISORIES.md)

How to use this folder:

1. Read source notes.
2. Extract one candidate action.
3. Decide: adopt now, backlog, or reject.
4. Link decision to repo artifact (plan/doc/test/workflow).

Keep this baseline intentionally short. Add new sources only when they are repeatedly useful.
