# Release 1.5.1

**Release date:** 2026-03

Build release following **1.5.0**: version bump, documentation and lint alignment, no behaviour change. All 146 tests pass with no errors or warnings.

## Changes since 1.5.0

- **Version:** 1.5.1 in `pyproject.toml`, `core/about.py` fallback, man pages, and docs (README, DEPLOY, PLANS_TODO).
- **Documentation:** Docs moved to the docs tree (mascot and logo-candidates content in `docs/mascot.md`, `docs/LOGO_CANDIDATES.md`); roaming READMEs under `api/static/` removed. Location policy clarified in docs/README (EN and pt-BR).
- **Lint:** Markdown and SonarQube alignment (MD028 blockquote, MD060 table style; `fix_markdown_sonar.py` S3776/S1481/S6326 fixes). No remaining markdown or script lint warnings in project files.
- **Docker:** Branded image **`fabioleitao/data_boar`** on Docker Hub: `fabioleitao/data_boar:latest` and `fabioleitao/data_boar:1.5.1`.

## How to get 1.5.1

- **From source:** `uv sync` or `pip install -e .` after pulling the tag or branch.
- **Docker (branded):** `docker pull fabioleitao/data_boar:1.5.1` or `docker pull fabioleitao/data_boar:latest`

## Building and pushing the branded Docker image

From the repo root, after tests pass:

```bash
uv run pytest -v -W error
docker build -t fabioleitao/data_boar:latest -t fabioleitao/data_boar:1.5.1 .
docker login
docker push fabioleitao/data_boar:latest
docker push fabioleitao/data_boar:1.5.1
```

Optionally push the same image under the legacy name for compatibility:

```bash
docker tag fabioleitao/data_boar:1.5.1 fabioleitao/python3-lgpd-crawler:1.5.1
docker push fabioleitao/python3-lgpd-crawler:1.5.1
```
