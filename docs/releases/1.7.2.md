# Release 1.7.2 (beta / in development)

**Status:** **`1.7.2-beta`** on `main` — pre-release line; **published** latest on GitHub/Docker may still be **1.7.1** until this bundle ships and is tagged.

**Version string:** See root **`pyproject.toml`** — follow [VERSIONING.md](../VERSIONING.md).

## Highlights (when this bundle ships)

- **WebAuthn JSON core (Phase 1a):** Optional **`api.webauthn`** + **`DATA_BOAR_WEBAUTHN_TOKEN_SECRET`** enable vendor-neutral **FIDO2 / passkey** JSON endpoints under **`/auth/webauthn/`** (PyPI **`webauthn`**), SQLite **`webauthn_credentials`**, signed session cookie — [ADR 0033](../adr/0033-webauthn-open-relying-party-json-endpoints.md). **Default:** feature **off**. **Not included:** HTML dashboard login enforcement (**[#86](https://github.com/FabioLeitao/data-boar/issues/86)** Phase 1b+).
- **Documentation:** Operator runbooks and **USAGE** / **TECH_GUIDE** describe **`api.webauthn`**; internal plan cross-links stay in **`docs/plans/`** (not linked from this release file per ADR 0004 external-tier rules).
