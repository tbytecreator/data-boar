# Evidence snapshots (template — operator tree)

**Real path:** `docs/private/social_drafts/drafts/evidence/` (gitignored from public GitHub `origin`).

**Purpose:** When you publish **manually** or **off-plan** and want a **durable copy** of what went live (for history and assistant context), add:

`YYYY-MM-DD_evidence_<network>_<short-slug>.md`

**Contents:** title, public URL, locale, short body (paste or summary). **No** secrets; redact PII unless this file stays strictly private per your policy.

**Hub:** always add or update the row in **`editorial/SOCIAL_HUB.md`** first — evidence files are **optional** supplements.

See **`docs/ops/today-mode/SOCIAL_PUBLISH_AND_TODAY_MODE.md`**.
